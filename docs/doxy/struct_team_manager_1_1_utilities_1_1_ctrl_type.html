Detailed Description

Type defintion for the id's of the CtrlType that can occur whenever the console closes. The event type we get back from the NativeMethods.SetConsoleCtrlHandler that registers to the kernal of the windows api a method callback in Program.CloseCallback. For more information, please see: https://docs.microsoft.com/en-us/windows/console/setconsolectrlhandler
