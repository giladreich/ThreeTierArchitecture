Detailed Description

The AscendingStrategy implementation will retrieve the data from the StrategyBase.DbLayer in ascending order. The constructor that gets the DatabaseType will pass forward to the base constructor in order to initialize the IDataLayer which is used as the StrategyBase.DbLayer. For more details, please see IStrategy documentation.
