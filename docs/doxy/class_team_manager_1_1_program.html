◆ HookConsole()

static void TeamManager.Program.HookConsole ( )
staticprivate

Hooks the console using the NativeMethods class methods that are exported from the windows api dlls in order to allow the console to show even when the project output type is set to Windows Application, that way we can swap between Gui or Tui. Also registers a callback method with the NativeMethods.SetConsoleCtrlHandler that will register a close event to the console even if someones click the X button on the top right side so we can do last wrapping before console closes.
